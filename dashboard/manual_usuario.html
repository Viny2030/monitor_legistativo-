📖 ¿Qué es el Monitor Legislativo?

El Monitor Legislativo — Cámara de Diputados es una herramienta de análisis de código abierto que mide la composición y el desempeño de los 257 diputados de la Nación Argentina. Organiza 12 indicadores en 4 dimensiones: Costos e Insumos, Eficiencia en el Proceso, Desempeño y Calidad, y Transparencia y Datos.

Todos los cálculos se realizan en el navegador, sin servidores externos. Los datos de composición se actualizan editando el array DIPUTADOS en los archivos HTML. No requiere instalación ni conexión a internet para funcionar.

🎯 Objetivo

Promover la transparencia legislativa y el debate informado sobre la composición y el funcionamiento de la Cámara de Diputados, poniendo datos oficiales en formato accesible para ciudadanos, periodistas e investigadores.

Navegación entre páginas

Todas las páginas comparten el mismo header de navegación (arriba a la derecha) y una barra de navegación interna (sticky, en azul) con acceso rápido a las secciones de cada página.

💡 En móvil, el header y la nav interna se adaptan automáticamente. Si los botones no caben, se reorganizan en varias filas.

Buscadores y filtros

1
Buscador de nómina — en la página Nómina, escribí nombre, bloque o distrito para filtrar los 257 diputados en tiempo real.
2
Filtros de detalle — en la página Detalle usá los selectores de bloque, distrito y mandato combinados para acotar la búsqueda.
3
Ordenar tabla de bloques — en la página Por Bloque hacé clic en cualquier encabezado de columna para ordenar ascendente o descendente.
4
Vista cards / lista — en la página Detalle podés alternar entre la vista de cards visuales y la tabla compacta con el botón ▦/☰.

Indicadores de composición (calculados en tiempo real)

CódigoNombreCómo leerlo
NEPNúmero Efectivo de PartidosCuántos partidos "efectivos" hay ponderando su tamaño. NEP <2,5 = baja fragmentación · 2,5–4 = moderada · >4 = alta. Se recalcula automáticamente.
IFÍndice de FragmentaciónVaría entre 0 (un solo bloque) y 1 (máxima dispersión). Complementa al NEP.
IRBTasa de Renovación BienalProporción de bancas renovadas en dic. 2025. La CN establece ~50% cada 2 años (Art. 50).
IRGRepresentación GeográficaIRG >1 = distrito sobre-representado · <1 = sub-representado. El mínimo de 5 bancas favorece a provincias pequeñas.

🔄 Cómo actualizar los datos

Actualización automática con el scraper

1
Clonar o actualizar el repositorio
git clone https://github.com/Viny2030/monitor_legistativo.git
2
Instalar dependencias
pip install -r requirements.txt
3
Ejecutar el scraper
python scraper_diputados.py
Genera el archivo nomina_diputados.csv con la nómina actualizada desde diputados.gov.ar.
4
Reemplazar el array DIPUTADOS en los archivos HTML con los nuevos datos del CSV. Los indicadores se recalculan automáticamente al recargar.
💡 Los indicadores NEP, IF, IRB e IRG se recalculan solos al modificar el array. No hace falta cambiar ningún otro valor.

Subir cambios a GitHub

git add .
git commit -m "Actualización nómina marzo 2026"
git push origin main

⚠️ Limitaciones y pendientes

Versión actual (v1.0)

La versión 1.0 cubre exclusivamente composición y representatividad. Los indicadores de actividad parlamentaria individual (proyectos presentados, votaciones nominales por diputado, comisiones integradas) requieren integración con el SIL — Sistema de Información Legislativa, prevista para la versión 1.1.

Limitaciones metodológicas conocidas

NEP: trata a todos los bloques como entidades separadas sin considerar afinidades ideológicas o acuerdos de coalición. Un NEP alto no implica necesariamente dificultad para formar mayorías.

NAPE: mide asistencia a sesiones, no presencia en todas las votaciones dentro de cada sesión. Un diputado puede constar como presente pero haberse retirado antes de votar.

RLS: la clasificación sustantivo/declarativo es propia del monitor y puede diferir de otras clasificaciones. Los acuerdos internacionales se clasifican como no sustantivos en esta versión.

Normalización de bloques: refleja la denominación oficial HCDN al momento de la consulta. Los bloques pueden cambiar de nombre, fusionarse o dividirse durante el período legislativo.

📖 Glosario rápido

NEP
Número Efectivo de Partidos — mide cuántos bloques "efectivos" hay, ponderando su tamaño.
IF
Índice de Fragmentación de Rae — dispersión del poder entre bloques (0 a 1).
IRB
Tasa de Renovación Bienal — porcentaje de bancas renovadas en la última elección.
IRG
Índice de Representación Geográfica — sobre/sub-representación de cada distrito.
CRC
Costo de Representación por Ciudadano — presupuesto HCDN dividido por la población.
RPS
Ratio de Profesionalización del Staff — planta permanente sobre total de empleados.
NAPE
Nivel de Asistencia y Permanencia Efectiva — % de diputados con asistencia perfecta.
COLS
Costo Operativo por Ley Sancionada — presupuesto dividido por leyes aprobadas.
RLS
Ratio de Legislación Sustantiva — % de leyes de política pública sobre total.
ECO
Efectividad del Control — % de auditorías AGN con seguimiento legislativo efectivo.
IAD
Índice de Accesibilidad Documental — velocidad de publicación de actas (escala 1–5).
TVD
Tasa de Veracidad de Datos — precisión del pipeline de scraping del monitor.
SIL
Sistema de Información Legislativa — base de datos oficial de proyectos y votaciones.
OPC
Oficina de Presupuesto del Congreso — organismo técnico de seguimiento presupuestario.
HCDN
Honorable Cámara de Diputados de la Nación.
AGN
Auditoría General de la Nación — control externo del sector público nacional.
